1. Откройте страницу просмотра видео вконтакте: Страница просмотра видео ВКонтакте (Откроется в новом окне)
  2. Скопируйте текст из нижепредставленного поля (правой кнопкой мыши на текст -> Копировать)
  3. Вставьте скопированный текст (клавиши Ctrl + V) в адресную строку на странице просмотра видео "В Контакте" (в адресной строке должно быть написано <$watch$>).
  4. Затем нажмите клавишу "Enter" и на страничке просмотра видео "В Контакте" появится окошко со ссылкой для сохранения файла. Вам останется только нажать на эту ссылку.
Для просмотра скачанного видео потребуется плеер для flv.
